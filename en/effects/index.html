# Effect development

Hyperion provides a powerful API to write your own effects, along with possible options and user interface to tune them.

  • Effect Files

# Effect Files

An effect has 3 different files.

File language Comment
neweffect.py Python (opens new window) The heart of the effect
neweffect.json JSON (opens new window) Contains options for the python file, which makes it configurable
neweffect.schema.json JSON Schema (opens new window) Creates the options UI and is used to validate user input. Read more

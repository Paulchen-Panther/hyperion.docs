# SK6812

  • 3 Pins
  • Ein Draht SPI-Pin (DAT)
  • Bildwiederholrate von 400Hz, eingeschränkte Datenrate von 800Hz
  • RGB-Variante ist dem WS2812B sehr ähnlich
  • Auch in RGBW-Versionen erhältlich
